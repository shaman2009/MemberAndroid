MemberAndroid
=============

Android client of memberapp

[Server](https://github.com/zzp8164/MemberWebservice) 



Build instructions
------------------

Dependency:

- Gradle 1.10
- Android Studio 0.4.3

Setup:

    $cd <your_project_dir>
    $git submodule init && git submodule update

Debug builds:

    $cd <your_project_dir>
    $gradle assembleDebug
